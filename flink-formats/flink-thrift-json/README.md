## thrift-json format

in our production kafka, ad server engine log is thrift protocol. when using it as flink table, we can create table as thrift-json format using the `FlinkTableSchemaGenerator`:


```
mvn clean package
```
```
$ java -jar  flink-formats/flink-thrift-json/target/flink-thrift-json-0.1-SNAPSHOT-test-jar-with-dependencies.jar 
Usage: net.qihoo.ads.flink.formats.thriftjson.util.FlinkTableSchemaGenerator: <table-name> <topic> <thrift-class> <kafka-bootstrap-servers>
```
```
$ java -jar  flink-formats/flink-thrift-json/target/flink-thrift-json-0.1-SNAPSHOT-test-jar-with-dependencies.jar 
Usage: net.qihoo.ads.flink.formats.thriftjson.util.FlinkTableSchemaGenerator: <table-name> <topic> <thrift-class> <kafka-bootstrap-servers>
```
